Debian

Ok so first up is Debian. Debian is one of the most popular linux distros, because majority of the linux distros you see will be based of of debian like Kali Linux, Ubuntu etc. Debian is a very lightweight distro. They update it a lot less, to keep high stability, but don't think it is not as secure because of that, because almost all the popular distros are more secure than windows! It requires a minimum of 512mb of ram, and 4gb hard drive space. Now don't let the requirments fool you! Debian has tons of features and is very user friendly, making it good for beginers and experts alike. Debian also has some of the most software support on linux. You can also pair it with

Wine

Wine which stands for wine is not an emulator, alows for vast software support. It allows you to double click executables even on linux. Back on track now Debian is a very good linux distro for beginers and experts alike, also offering great security.




Ubuntu

Ubuntu is widely considered one of the most popular linux distros, for it's ease of use software support and security. Ubuntu is a Debian based distro. The system requirments are a little bit higher 4gb ram and 25gb storage space, but don't be scared away by the higher requirments. It is still a great distro. Their update model is very fast frequent updates. They add many features frequently and many security patches. They are ambitious with their updates and they are great, but they loose some stability, compared to other distros like Debian. Don't let this fool you though it is still a very good distro.
